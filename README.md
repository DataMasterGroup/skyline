# Skyline

**Skyline** is a blockchain built using Cosmos SDK and Tendermint.

- [Whitepaper](http://www.dmchain.io)

## Quick Start

### Build

``` shell
# Build only to ./build/

$ make build

# Build & install

$ make install

# Run all test cases & build & install

$ make all
```

### Start

``` shell
./scripts/start_standalone.sh
```
